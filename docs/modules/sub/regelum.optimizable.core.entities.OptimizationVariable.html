regelum.optimizable.core.entities.OptimizationVariable

class regelum.optimizable.core.entities.OptimizationVariable(name: str, dims: tuple, data: Optional[Any] = None, metadata: Optional[Any] = None, is_constant: bool = False, hooks: regelum.optimizable.core.entities.ChainedHook = <factory>)

Represent an optimization variable within the optimization process.

This base class encapsulates data and metadata for variables used in optimization, allowing them to be treated as either constants or decision variables.

Parameters

  • name – The name of the variable.

  • dims – The dimensions of the variable as a tuple.

  • data – The initial data for the variable. Defaults to None.

  • metadata – The metadata associated with the variable. Defaults to None.

  • is_constant – Flag indicating if the variable is a constant. Defaults to False.

  • hooks – A ChainedHook instance containing hooks to be applied to the variable. Defaults to an empty ChainedHook.
