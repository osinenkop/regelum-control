regelum.objective.reinforce_objective

regelum.objective.reinforce_objective(policy_model: regelum.model.PerceptronWithTruncatedNormalNoise, observations: torch.FloatTensor, actions: torch.FloatTensor, tail_total_objectives: torch.FloatTensor, total_objectives: torch.FloatTensor, baselines: torch.FloatTensor, is_with_baseline: bool, is_do_not_let_the_past_distract_you: bool, N_episodes: int) → torch.FloatTensor

Calculate the surrogate objective for REINFORCE algorithm.

Сalculates the following surrogate objective: \(\\frac{1}{M}\\sum_{j=1}^M \\sum_{k=0}^N \\log \\rho_{\\theta}(y_{k|j}, u_{k|j} \\left(\\sum_{k'=f_k}^N \\gamma^{k'} r(y_{k|j}, u_{k|j}) - B_k\\right),\) where \(y_{k|j}\) is the observation at time \(k\) in episode \(j\), \(u_{k|j}\) is the action at time \(k\) in episode \(j\), \(\\rho_{\\theta}(u \\mid y)\) is the probability density function of the policy model, \(f_k\) is equal to \(k\) if is_do_not_let_the_past_distract_you is True, and \(f_k\) is equal to 0 if is_do_not_let_the_past_distract_you is False, :math: B_k is the baseline, which equals 0 if is_with_baseline is False and the total objective from previous iteration if is_with_baseline is True, :math: M is the number of episodes, \(N\) is the number of actions.

Parameters

  • policy_model (GaussianPDFModel) – The policy model used to calculate the log probabilities.

  • observations (torch.FloatTensor) – The observations tensor.

  • actions (torch.FloatTensor) – The actions tensor.

  • tail_total_objectives (torch.FloatTensor) – The tail total objectives tensor.

  • total_objectives (torch.FloatTensor) – The total objectives tensor.

  • baselines (torch.FloatTensor) – The baselines tensor.

  • is_with_baseline (bool) – Flag indicating whether to subtract baselines from the target objectives.

  • is_do_not_let_the_past_distract_you (bool) – Flag indicating whether to use tail total objectives.

  • device (Union[str, torch.device]) – The device to use for the calculations.

  • N_episodes (int) – The number of episodes.

Returns

surrogate objective value.

Return type

torch.FloatTensor
