class regelum.optimizable.core.entities.NestedFunction(name: str, dims: tuple, data: Optional[Any] = None, metadata: Optional[Any] = None, is_constant: bool = False, hooks: regelum.optimizable.core.entities.ChainedHook = <factory>, nested_variables: regelum.optimizable.core.entities.VarContainer = <factory>)

Represent a nested function within the optimization process.

This class extends the OptimizationVariable to represent a nested function, including variables on which it depends.

Parameters

nested_variables – A container for variables that the nested function depends on. Defaults to an empty VarContainer.
