The notations used throughout this document are defined as follows:

  1. \(\policy _0\) is stabilzing policy, i. e., the policy if \(\action _t = \policy _0(\state _t)\) implies that the distance between \(\state _t\) and \(\G \) vanishes over time, where \(\G \) is some given neighbourhood of the origin

  2. \(\hat \kappa _\low \), \(\hat \kappa _\up \) is a pair of \(\Kinf \) functions. In other words, \(\hat \kappa _\low \), \(\hat \kappa _\up \) satisfy: \begin {align*} & \hat \kappa _\low (0) = \hat \kappa _\up (0) = 0, \\ & \lim \limits _{\state \ra \infty } \hat \kappa _\low (\state ) = \lim \limits _{\state \ra \infty } \hat \kappa _\up (\state ) = \infty . \end {align*}

  3. \(\hat \Value ^w\) is a state-valued critic model (\(w \in \W \), where \(\W \) is compact and \(0 \notin \W \))

  4. \(\overline {\nu } \in \R _{>0}\) a desirable Lyapunov decay rate
