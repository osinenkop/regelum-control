When creating hyperlinks, it is advisable to use the \hyperlink command to ensure optimal functionality on the website. For instance, to create a hyperlink to current subsection, one would use: 

\hyperlink{links-sections-and-subsections}{current subsection}

Similarly, to link to the previous subsection: 

\hyperlink{sections-and-subsections}{previous subsection}

Although these links will not be functional within a PDF, they will operate correctly on the website. For accurate linking, follow these steps:

  • Identify the name of the section or subsection to which you wish to link, such as

    Links. Sections and subsections

  • Strip all punctuation from it to yield

    Links Sections and subsections

  • Convert the text to lowercase:

    links sections and subsections

  • Replace spaces with hyphens to form the link identifier:

    links-sections-and-subsections

  • Insert this identifier into the first argument of the ∖hyperlink command: 

          \hyperlink{links-sections-and-subsections}{a hyperlinked text}
