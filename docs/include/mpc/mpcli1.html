Consider the following optimal control problem: \[ \sum _{t = 0}^{T-1}\cost (\state _{t}, \action _{t}) \ra \min _{a_0, \ldots , a_{T-1}}, \] where

  1. The state at each time step \(t \in \{1, \ldots , T-1\}\) is derived by applying the transition function \(\transit : \states \times \actions \to \states \): \[ \state _{t} = \transit (\state _{t-1}, \action _{t-1}), \] with the initial state \(\state _0\) given.

  2. The function \(c(\state , \action )\) represents the cost associated with the current state \(\state \) and action \(\action \)
