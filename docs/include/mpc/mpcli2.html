Input: horizon \(N \in \N \)
  for Time step \(t := 0 \dots T-1\) do
  Solve \[ \begin {array}{ll} \min \limits _{\action _t, \dots , \action _{t+N-1}} & \sum \limits _{t^\prime =t}^{t+N-1} \cost ( \state _{t^\prime }, \action _{t^\prime } ) \\ & \sut \forall t^\prime \in [t:t+N] \spc \state _{t^\prime } \in \states \\ & \hphantom {\sut } \forall t^\prime \in [t:t+N] \spc \action _{t^\prime } \in \actions \end {array} \]
  Apply \(\action _t\), the first action from the solution sequence: \(\state _{t+1} = \transit (\state _t, \action _t)\)
  end for
