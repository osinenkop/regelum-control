You can define your table by utilizing table environment: 

\begin{table}[H]
    \caption{Table caption}
    \label{table:my-table}
    \begin{tabularx}{0.97\textwidth}{ |p{0.3\textwidth}|p{0.6\textwidth}| }
        \hline
        $Q$ & Quality function \\
        $r$ & Running objective \\
        \hline
    \end{tabularx}
\end{table}
