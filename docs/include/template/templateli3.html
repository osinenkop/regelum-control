Sections and subsections can be delineated using the following commands: 

\section*{...} \subsection*{...}

For example, this section was initiated with: 

\section*{Sections and subsections}

And the subsequent subsection was generated with 

\section*{Links. Sections and subsections}

It is important to use the asterisk * with these commands, such as \section*{...} or \subsection*{...}, to prevent the automatic numbering of sections.
