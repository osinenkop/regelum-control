You can define your figure by utilizing figure environment. Thus, the code 

\begin{figure}[H]
    \caption{My awesome figure!!!}
    \label{fig:my-fig}
    \includegraphics{img.pdf}
\end{figure}

provides us

Please ensure that the file img.pdf is located within the docs/tex/gfx directory, as indicated by the path: docs/tex/gfx/img.pdf
